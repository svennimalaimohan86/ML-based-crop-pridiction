# 🌾 ML-Based Crop Prediction & Disease Detection

## 📌 Overview
This project uses **Machine Learning and Deep Learning** to:
- Predict crop risk based on soil and weather data
- Detect plant diseases using leaf images

It helps farmers make better decisions and improve crop yield.

---

## 🚀 Features

### 🌱 Crop Risk Prediction (ML)
- Uses N, P, K, temperature, humidity, pH, rainfall
- Predicts:
  - Low Risk
  - Medium Risk
  - High Risk
- Suggests pesticide actions

---

### 🌿 Disease Detection (Deep Learning)
- Uses leaf images
- Detects diseases like:
  - Blight
  - Rust
  - Healthy
- Provides prediction with confidence score

---

## 🧠 Technologies Used

- Python 🐍
- Scikit-learn (Machine Learning)
- TensorFlow / Keras (Deep Learning)
- Pandas, NumPy
- Matplotlib (Visualization)
